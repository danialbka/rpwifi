Republic Polytechnic WiFi Setup

Quick Fix Guide

1

Set DNS to Automatic

Windows:

  1. Open Settings → Network & Internet
  2. Click Wi-Fi → Select your network
  3. Click Edit under IP settings
  4. Find DNS settings and change to Automatic
  5. Click Save

Mac:

  1. Open System Preferences → Network
  2. Select Wi-Fi → Click Advanced
  3. Go to DNS tab
  4. Remove any custom DNS servers
  5. Click OK and Apply
2

Connect to POLITE-MALL

  1. Open your WiFi settings
  2. Look for the network named POLITE-MALL
  3. Connect to it (no password required)
  4. Wait for connection to establish
3

Test Your Connection

Your WiFi should now be working! Try accessing:

  • Any website in your browser
  • Online services

Still Having Issues?

  • Make sure you've disconnected from other WiFi networks
  • Restart your device if the connection doesn't work
  • Forget the POLITE-MALL network and reconnect
  • Contact IT Support if problems persist
